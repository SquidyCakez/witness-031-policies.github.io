# Witness-031 Privacy Policy

Last updated: June 13, 2026

Witness-031 is a private Discord bot for Roblox development operations.

The bot may process:
- Discord user IDs
- Discord role IDs
- Channel IDs
- Messages sent to configured bot channels
- Bug reports, deployment requests, and command history
- GitHub or Roblox deployment metadata, such as commits, branches, place IDs, and deploy logs

The bot does not intentionally collect passwords, payment information, Roblox account credentials, or Discord login credentials.

Data is used only to:
- Respond to authorized Discord requests
- Triage bugs and balance issues
- Prepare or log Roblox hotfixes and deployments
- Enforce role-based permissions
- Maintain audit logs

Some implementation and deployment activity may be logged to the configured Discord logging channel.

Data may be stored in OpenClaw logs, local bot configuration, Discord message history, and related development tools. To request review or deletion of bot-related data, contact the owner through the Discord server.

API keys, tokens, and secrets are not intentionally exposed to users and should be stored only in environment variables or secure configuration.
